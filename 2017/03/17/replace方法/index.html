replace方法

Posted on 2017-03-17 Edited on 2022-02-10

replace() 方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子字符串。
语法 string.replace(searchvalue,newvalue)

参数 描述
searchvalue 必须。规定子字符串或要替换的模式的 RegExp 对象。
newvalue 必须。规定了替换文本或生成替换文本的函数。

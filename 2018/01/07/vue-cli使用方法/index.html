vue-cli使用方法

Posted on 2018-01-07 Edited on 2022-02-10

安装 vue-cli

使用npm install vue-cli -g来安装或者更新vue -cli。

初始化项目

vue init 模板名称 项目名称
这里的模板名称官方提供5种：webpack、webpack-simple、browserify、browserify-simple、simple。
项目名称自己取，不需要加引号，不能大写。
创建好项目后npm install下载依赖，然后npm run dev启动项目。

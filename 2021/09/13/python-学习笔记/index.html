nums1 = A 和 nums1[:] = A

nums1 = A:更改 nums1 这一变量名所指向的对象。让 nums1 变量指向 A 所指向的对象

nums1[:] = A:对 nums1 指向的对象赋值。把 A 变量指向的对象的值逐个 copy 到 nums1 指向的对象中并覆盖 nums1 指向的对象的原来值。
